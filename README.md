# Vakhtan
Russian draughts engine with GUI written in pure JavaScripti<br>
(Программа названа в честь <a href="https://www.youtube.com/@Vakhterov">Владимира Вахтерова</a>)

# Play now
https://maksimkorzh.github.io/vakhtan/

# For developers
 - bug free move generator (based on Kestog by by Kestutis Gasaitis), see perft.txt
 - Negamax search with alpha-beta pruning
 - Killer move ordering
 - Iterative deepening
 - Material + PST evaluation
 - Own GUI
 - Load position from FEN
